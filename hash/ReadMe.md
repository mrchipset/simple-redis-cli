# Hash
HASH is a hash dictionary key value set and get app.

## Usage
`hash.exe [Operate] [Dict] [Key] [Value]`

[Operate] : GET, SET, DEL, LIST

### GET
`hash.exe GET dict foo`

### SET
`hash.exe SET dict foo bar`

### LIST
list all the keys
`hash.exe LIST dict`

### DEL
delete specific key
`hash.exe DEL dict foo`
